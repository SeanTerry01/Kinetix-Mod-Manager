# Skyrim Access

Skyrim Access is an SKSE plugin that makes The Elder Scrolls V: Skyrim Special Edition (and Anniversary Edition / VR) playable by blind and visually impaired players. It uses the NVDA screen reader to voice nearly everything the game presents, and adds navigation aids so you can explore, fight, and complete quests by ear.

This is an offline summary bundled with Kinetix Mod Manager. The mod's authoritative documentation lives on its Nexus Mods page. When you are logged in to Nexus and connected to the internet, reopen this viewer to load the latest description straight from that page.

## Features

### Spoken Interface

Skyrim Access voices the game's menus (inventory, magic, journal, map, barter, lockpicking, crafting, and more), the HUD, on-screen subtitles, and your current stealth status, so you always know what the game is showing.

### Navigation and Combat Aids

- An object scanner that finds and describes nearby objects, NPCs, doors, and items.
- Bow auto-aim and enemy lock-on to make ranged and melee combat workable without sight.
- Autowalk for travelling along paths and between locations.
- Automatic solving of the vanilla puzzles that would otherwise require sight.

### Content Coverage

The full main quest, all of the major guild questlines (the Companions, the Dark Brotherhood, the Thieves Guild, and the College of Winterhold), the Daedric quests, and the official DLCs (Dawnguard, Dragonborn, and Hearthfire) are playable.

## Installation

1. Install SKSE (the Skyrim Script Extender) for your version of the game.
2. Install the NVDA screen reader, which Skyrim Access uses to speak.
3. Install Skyrim Access from its Nexus Mods page with Kinetix Mod Manager (or your preferred mod manager).
4. Make sure the SKSE plugin files land in the game's Data folder; Kinetix Mod Manager handles SKSE plugin placement for you.

## Keybindings

Skyrim Access exposes its keys through its in-game configuration (MCM), and many of them can be remapped there. Because they reflect your own setup, the most reliable way to review the current keys is from inside Kinetix Mod Manager: press Ctrl+H to open the Accessibility Controls viewer, which reads the live keybindings for your installed mods (including Skyrim Access) directly from MCM.
