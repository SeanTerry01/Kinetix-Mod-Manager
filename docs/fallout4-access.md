# Fallout 4 Access

Fallout 4 Access is an F4SE plugin that adds screen reader support to Fallout 4 for blind and visually impaired players. It speaks menu text, dialogue, and gameplay information through your screen reader so you can navigate the wasteland by ear. The mod is under active development.

This is an offline summary bundled with Kinetix Mod Manager. The mod's authoritative documentation lives on its Nexus Mods page. When you are logged in to Nexus and connected to the internet, reopen this viewer to load the latest description straight from that page.

## Features

### Spoken Interface

Fallout 4 Access works with the NVDA, JAWS, and SAPI screen readers to voice the game's menus, dialogue, the Pip-Boy, terminals, the V.A.T.S. targeting system, and the HUD, so you always know what is on screen.

### Navigation Aids

An object scanner helps you find and identify nearby objects, characters, and points of interest, making it possible to explore and loot without sight.

## Installation

1. Install F4SE (the Fallout 4 Script Extender).
2. Install a supported screen reader (NVDA, JAWS, or a SAPI voice).
3. Install Fallout 4 Access from its Nexus Mods page with Kinetix Mod Manager (or your preferred mod manager).
4. Make sure the F4SE plugin files land in the game's Data folder; Kinetix Mod Manager handles F4SE plugin placement for you.

## Keybindings

Fallout 4 Access exposes its keys through its in-game configuration (MCM), and many of them can be remapped there. Because they reflect your own setup, the most reliable way to review the current keys is from inside Kinetix Mod Manager: press Ctrl+H to open the Accessibility Controls viewer, which reads the live keybindings for your installed mods (including Fallout 4 Access) directly from MCM.
